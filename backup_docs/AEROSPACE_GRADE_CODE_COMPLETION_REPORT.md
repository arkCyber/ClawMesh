## 🆕 新增功能模块

### 1️⃣ 实时通信系统 (`clawmesh_realtime`)

#### 📦 模块结构
```
crates/clawmesh/realtime/
├── Cargo.toml          # 依赖配置
├── README.md           # 模块文档
└── src/
    ├── lib.rs          # 模块入口
    ├── connection.rs   # WebSocket 连接处理
    ├── session.rs      # 会话管理
    ├── manager.rs      # 连接和房间管理器
    └── messages.rs     # 消息类型定义
```

#### ✨ 核心功能

**WebSocket 连接管理**
- 支持多连接并发（每用户最多 5 个连接）
- 心跳检测机制（30 秒间隔）
- 连接超时保护（5 分钟）
- 自动重连支持

**用户在线状态**
```rust
pub enum PresenceStatus {
    Online,   // 在线
    Away,     // 离开
    Busy,     // 忙碌
    Offline,  // 离线
}
```

**房间管理**
- 动态房间创建/销毁
- 成员加入/离开
- 消息广播
- 房间权限控制

**消息类型**
- `text` - 文本消息
- `notification` - 系统通知
- `presence` - 状态更新
- `typing` - 输入指示器
- `ping/pong` - 心跳
- `join/leave` - 房间操作
- `error` - 错误消息

#### 🏗️ 架构设计

```
┌─────────────────────────────────────────┐
│         WebSocket Endpoint              │
│  GET /ws - 建立 WebSocket 连接          │
├─────────────────────────────────────────┤
│         WsSession (Actix Actor)         │
│  - 心跳监控 (30s interval)              │
│  - 消息处理和路由                        │
│  - 连接生命周期管理                      │

│      ConnectionManager                  │
│  - 连接注册/注销                         │
│  - 用户在线状态跟踪                      │
│  - 多连接支持 (max 5/user)              │
├─────────────────────────────────────────┤
│         RoomManager                     │
│  - 房间成员管理                          │
│  - 消息广播到房间                        │
│  - 房间权限验证                          │
└─────────────────────────────────────────┘
```

        // 其他角色按层级检查

    }
}
