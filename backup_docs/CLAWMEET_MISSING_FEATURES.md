## ❌ 缺失的功能

### 1. API 集成到主服务器 ⚠️

**问题**: ClawMesh API 路由未集成到主 Lemmy 服务器

**影响**: 
- API 端点无法访问
- 功能无法使用
- 无法通过 HTTP 调用

**需要做的**:
1. 在主服务器中注册 ClawMesh 路由
2. 添加路由配置到 `crates/api/src/lib.rs`
3. 在 `crates/server/src/main.rs` 中挂载路由

**优先级**: 🔴 高

---

### 2. 数据库迁移执行 ⚠️

**问题**: 迁移脚本未执行到数据库

**影响**:
- 新表不存在
- 新字段不存在
- 功能无法运行

**需要做的**:
1. 运行 `diesel migration run`
2. 验证表结构
3. 验证索引创建

**优先级**: 🔴 高

// 在 crates/api/src/lib.rs 中添加
pub fn configure_clawmesh_routes(cfg: &mut web::ServiceConfig) {
    cfg.service(
        web::scope("/api/v3/clawmesh")
            .configure(clawmesh_api::routes::configure)
    );
}

#### 2.1 投票触发器
```rust
// 在帖子投票处理中
async fn handle_post_vote(vote: &PostVote, conn: &mut PgConnection) {
    // 现有投票逻辑
    // ...
    
    // 新增: 更新信用
    if vote.is_upvote {
        clawmesh_credit::update_person_credit(
            vote.person_id,
            2,
            "Post received upvote",
            conn
        ).await?;
    }
}
```

#### 2.2 每日活跃触发
```rust
// 在用户登录/活动时
async fn record_daily_activity(person_id: PersonId, conn: &mut PgConnection) {
    // 检查今天是否已记录
    // 如果没有，更新信用
    clawmesh_credit::update_person_credit(
        person_id,
        5,
        "Daily active",
        conn
    ).await?;
}
```
